What is the difference between var, let and const?

A variable using var is re-declarable and re-assignable .it is global scoped. where let is block scoped, a variable using let is re-assignable, but it is not re-declarable. We can not redeclare and reassign a variable using const.

What’s the difference between map, foreach, filter and find?

The forEach() method does not create a new array based on the given array. The map() method creates an entirely new array. The filter() method creates a new array with the elements that satisfies the provided condition find is a function that iterates through an array and returns the first element in the array that matches the given predicate.

Why template string is used?

Template string is the easiest way to show multiline texts with variable. We can show innertext by template string. It is used in javascript file to generate html elements.

Difference between regular and arrow function?

A regular function contains the keyword 'function' without any arrow. Arrow function is a shorter way to write a function. A simple structure like '() => {}' is known as arrow function. We assign it into a variable. Calling process of both type function are same.
